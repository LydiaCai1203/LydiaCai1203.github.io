政治是什么

Reading note of What’s Politics

January 18, 2023
2023 摘抄

1. 政治的起源

一个事实是：人类很早就开始有集体行动的特征，他们喜欢一大堆人集中在一个约略特定的地域范围里活动，然后以某种分工合作的方式彼此照顾，并渐渐发展出简单的家庭单位和社会组织结构。

1.1 集体行动的原因

1. 资源的稀缺性
    a. 地球自然环境本身具备的 “绝对稀缺性”（水、石油等）
    b. 人为因素带来的分配不均（强者垄断导引发的稀少状态）
2. 生存利益
3. 竞争
    由于物种的存续需要大量资源，而绝大多数的资源具有稀缺性，因此高度竞争就成为一种自然而然的现象，通过团结模式来运作的聚居行为，至少是动物在面对激烈竞争时必须选择的理性结果

1.2 社会的成因

1. 部落（5000 ～ 8000 年前）
那一部分因自然选择而养成群居习惯的人类，从游猎逐渐发展成了具有定居性质的 农耕畜牧文化，部落就开始出现了。人类由此具有创造自然的能力（人为加工增加动植物产量），由此强化了自身的竞争力。

2. 定居带来的分工制度
定居后人类由 共谋安全 转为 合作为主，群体规模日益增长，由此导致生活形态更为复杂，于是分工制度出现了。
+ 如何分工（给谁分配什么职业）
+ 分配问题（公平地取长补短、各取所需）
+ 维持既有分工状态（让大家都愿意长期共同定居生活的前提下，如何让社会制度稳定）

3. 政府 && 无组织状态
人类早期如何究竟是如何开始组织生活的？并无历史证据证明。但有两派观点：
+ 政府 - 欧洲 霍布斯
生存的要务导致 人人为战，人人为敌。必然是存在一套权威体系用以对抗人类野蛮的天性。
+ 无组织状态 - 英国 洛克
天赋人权，个人自由且财产不受侵犯，也不该受组织管束。

人类早期自保能力的缺乏，在面对自然，组织这种合众人力量机制有切实存在的必要。组织存在的目的已经从 合理安排分工 转为 谋求更高福利。

1.3 制度

1. 制度设计的前提 - 效率
即能否快速有效地解决大家的需求问题，是群体能否承受生存考验的前提。首先必须把少数有决策能力的人找出来，赋予其领导角色，然后环绕决策者形成所谓的政府架构，主要的目的是发掘社会上的各种需求。最后为了有效地 发掘、分析、决定需求的优先级，政府内部必须适时地进行一定程度的分工架构重组。
+ 角度分化现象
为求取群体所需的最大公约数，需要 区分出 决策者、一般大众。
+ 政治阶层现象
通常分为 决策者（最先出现）、执行者、被统治者。建筑在法律规范基础上的权力关系，从追求效率的层面上看无可厚非。
+ 组织分工现象
根据定居生活的各种复杂条件以及社会发展现实，人们必须先区分具有各种不同特征的工作。这种分工现象的精细程度与社会生活的复杂度有关（历史进展、文明程度、政府角色的变迁、人口数）。

2. 制度设计的重要因素 - 从人性出发 && 制衡原则
+ 人性
人性往往善恶兼具 - 作者。
尽是些忘恩负义与反复无常的伪君子，除非形势所迫，理论上绝不会主动为善。 - 马基雅维利
权力使人腐化，绝对的权力使人绝对的腐化。因为人类总是习惯根据结果来判定行为的是非对错。善良的人也要根据情形适度为恶。 - 阿克顿爵士

+ 制衡原则
只要保障君权，可以大幅削弱保障民权。
+ 保障君权不受侵犯
文武分治。
+ 保障民权不受侵犯
立法、司法、行政 三权分立制度。

3. 责任政治 - 规范政客
责任分为三种：政治责任、法律责任、道德责任。决策者三种责任都要负，执行者负后两者责任。现实被问责的往往是执行者，决策者却逍遥在民意之外。
+ 言行不一、虚假广告（政客需要以其政见而吸引选票
+ 无法满足公众需求（无能无才

责任政治只是种 “自由心证” 的消极心理机制，且仅存在于民主机制中，从历史来看政务并无法因此而受到监督。（所谓 自由心证 是指：法官斟酌全辨论意旨及调查证据之结果后，依伦理及经验法则判断事实之真伪的过程）

8. 现状
+ 推卸责任
一些国家面对政济危机时，处罚低阶官员或随便捉些高官下台搪塞政治责任。
+ 转移焦点
美国的克林顿总统，遭遇性丑闻缠身时，发动对伊拉克的空袭以转移民众注意。
+ 寻找代罪羔羊
钟南山。

2. 领袖是怎么被筛选出来的

2.1 寡头铁律 - 精英是如何脱颖而出

政治永远被少数人控制，只是少数精英在玩的游戏。
政治研究 其实是研究 何谓影响力以及谁具有影响力。

1. 客观因素：具备智识与领导统御能力等突出的个人特质
    + 智识
        不仅是教育程度，而是理解社会生活各个层面的常识
    + 领导才能
        不仅是一般的组织与处理问题的能力，而是 领袖魅力。
    + 总之是人们在某一时期所凝聚的共识，接近人们的期待，就会更容易被大多数人所接受。

2. 主观因素：谁也不会真正地强有力，除非他掌握了重要的机构

3. 总结
寡头铁律未必因为不符合 民主政治的多数统治原则 而带来政治上的不稳定；政治制度能否稳定运作与统治阶级能否取得 正当性，即人们服从统治的心理接受程度相关。

2.2 正当性问题 - 为什么是这群人

谁来统治并不重要，重要的是统治者能否反映并且满足社会的需求。现实不尽如人意，精英逐渐和人群脱节，甚至自我构建起一个具有相同心理基础的 社会阶级(贵族)，除了彼此接纳并信任自己创造的互动模式以外，还经常联姻通婚，希望通过生理和社会关系来巩固这个小圈子。那么为什么这种不正确的现象却一直在历史里存在呢？

1. 人民自己 - 容忍度太高
+ 人民参政意愿薄弱（且不论能力）
+ 落实政策的时间差（政客摆弄民众耐心的惯用手法）
+ 民众盲目崇拜偶像（政客利用现代媒体打造自我光环）

2. 政治精英 - 通过创造意识形态获得正当性
控制社会上主要物质力量的阶级 同时也掌握了 主要的知识力量，他们是能随心所欲主导心智活动的阶级，同时也掌握了控制心智活动的工具。
+ 对未来的期望和憧憬（天朝意象 或 人民万岁）
+ 合理化精英取得政治权力地位的途径（君授神权 或 万世一系）
+ 合理化解释精英做出的决策

3. 历史变迁过程 - 不断的同类相残
+ 拥有 作战技能 的精英会首先获得政治地位
+ 缜密组追能力的文人会取代军人获得政治地位
+ 终身制 && 世袭制 现象随之出现巩固（世袭制甚至通过对某个人能力的吹嘘和对其功绩的怀念，使人民相信他的儿子是最佳接班人）

现在所谓的 民主选举 并不完全民主，选择是否理性并不重要，重要的是被选出来的人能否满足人民的需求。目前看来结果不尽如人意。

2.3 权力 - 政治中最具争议的副产品

1. 概念
权力是一个人或一群人，在某些社会行动中，甚至在不顾其他参与者抵抗的情况下，实现自己意志的可能性。

2. 特征
+ 强制
强迫别人做事的能力
+ 垄断
希望只有自己拥有这种能力

3. 组成
+ 通过 战争 等手段实施强迫的过程
+ 多数人默认 领袖具有垄断性地位的心理过程（群居生物的生物性本能）
权力的目的就是取得对分配资源的优先或独占性发言权

4. 种类
+ 强制性权力
利用 物理制裁 或 暴力威胁 手段，让受影响的人为避免遭受痛苦而服从。
+ 象征性权力（更重要，毕竟金字塔顶端的是少数人，无法长期使用暴力制服）
通过 社会规范 与 意识形态 手段，特别是某种阶级观念的建立，让人们养成服从习惯，甚至是心底下承认政客的贡献而高人一等，巩固权力关系。

比如：
皇家进出宫廷时候富丽堂皇的仪仗，总统出门警车开道、保镖服侍。
其目的就是为了塑造政客高人一等的权力形象。
